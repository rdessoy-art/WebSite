# Google Apps Script Form Handler

This folder contains everything you need to set up Google Apps Script to handle your Fan Club form submissions.

## What's Included

- **Code.gs** - The main Google Apps Script code
- **SETUP-INSTRUCTIONS.md** - Detailed step-by-step setup guide
- **form-update-example.html** - Example of the updated form code

## What This Does

✅ Receives form submissions from your website
✅ Saves all submissions to a Google Sheet
✅ Sends you an email notification for each signup
✅ Sends a welcome email to new members
✅ Validates email addresses
✅ Timestamps all submissions

## Quick Start

1. Follow the instructions in **SETUP-INSTRUCTIONS.md**
2. The whole process takes about 10-15 minutes
3. No coding knowledge required!

## Benefits Over Web3Forms

- **Free forever** (no limits with Google Apps Script)
- **Full control** over your data in Google Sheets
- **Customizable emails** - make them match your brand
- **No monthly fees** or submission limits
- **Easy to export** data from Google Sheets
- **Can add integrations** later (MailChimp, CRM, etc.)

## Support

- The troubleshooting section in SETUP-INSTRUCTIONS.md
